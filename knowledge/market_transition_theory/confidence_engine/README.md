# MarketMirrorAI Confidence Engine



Purpose:



Determine the confidence level of a diagnosis.



OpenClaw may identify multiple candidate patterns.



The Confidence Engine ranks candidates.



Inputs:



- visual similarity

- rule confirmation

- volume confirmation

- timeframe confirmation

- SPY confirmation

- market context

- validation history



Output:



Primary Pattern



Secondary Pattern



Confidence Score



Decision Recommendation



Core Principle:



The highest confidence pattern is not automatically tradable.



Rules and confirmations must still agree.



Confidence Levels:



90-100

Very High



75-89

High



60-74

Moderate



Below 60

Low



Operational Rule:



When confidence is low, prefer WAIT.

# Page-Linked Visual Policy



Visual examples must not be treated as independent images.



Every visual reference must point to a complete page unit:



- page text

- page image

- page manifest



The correct reference format is:



Book:

Page:

Text File:

Image File:

Manifest File:



Operational Rule:



When OpenClaw uses a visual example, it must load the full page unit.



Visual interpretation must remain linked to the text explanation from the same page.



No visual example should be copied or interpreted without its original page context.



Purpose:



Prevent isolated chart interpretation and preserve the author's complete explanation.

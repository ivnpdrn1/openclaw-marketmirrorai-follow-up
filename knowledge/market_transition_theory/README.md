# Market Transition Theory



Objetivo:



Construir un sistema de detección temprana de:



- Cambios de tendencia alcista

- Cambios de tendencia bajista

- Continuación de tendencia

- Falsos rompimientos

- Acumulación

- Distribución

- Confirmaciones CALL

- Confirmaciones PUT

- Gestión de riesgo



Fuentes:



- CREA TU PROPIA RIQUEZA

- DESCUBRE EL MUNDO DE LAS INVERSIONES Y APRENDE A GANAR EN LA BOLSA DE VALORES

- ESTRATEGIAS AVANZADAS DE INVERSIÓN Y OPCIONES FINANCIERAS

- GANANDO EN CUALQUIER CONTEXTO Y DIRECCIÓN



Estructura:



books/      -> PDFs originales

extracted/  -> texto extraído

summaries/  -> conceptos resumidos

patterns/   -> patrones detectados

signals/    -> reglas operativas
